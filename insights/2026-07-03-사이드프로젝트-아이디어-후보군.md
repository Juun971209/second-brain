# 스터디 주제 후보 — 사이드프로젝트 아이디어 4종

> 출처: Notion — KFC UX 스터디 "주제선정" 브레인스토밍 (5/26~6/2, 준형/은지/지현 3인)
> 최종 KFC 앱 개선에 밀린 낙선 후보들. 아이디어 자체는 재사용 가치 있어 보존.
> 정리일: 2026-07-03

## 1. BRIEF 서비스 — AI 브랜드 브리프 생성기 (최종 투표 14점, 2위)

**도메인**: B2B · 크리에이티브 | **방향**: 신규기획

**선정 이유(문제 인식)**
에이전시 실무에서 결과물이 처음 기획과 전혀 다르게 끝나는 경우가 많음. 원인은 클라이언트와의 초기 커뮤니케이션 — 클라이언트가 브랜딩·디자인 언어 자체를 모름. 이건 클라이언트 성의 문제가 아니라 구조적 공백. 타입폼/구글폼 같은 정적 폼은 있지만 클라이언트가 어떻게 채워야 할지 몰라 빈칸이나 엉뚱한 답변으로 돌아옴.

**해결하려는 문제**
클라이언트는 브랜드 톤앤매너·타겟을 언어로 정리해본 경험 자체가 없음. 질문해도 애매한 답변, 미팅 3~5회 반복해도 방향 안 잡힘, 후반에야 수정 요청. → AI가 답변을 받아 다음 질문을 동적으로 결정하면, 클라이언트는 용어를 몰라도 대화하듯 답하고 결과가 구조화된 브리프로 완성됨.

**기대효과**
첫 미팅 전 브리프 완성 → 방향 잡힌 채 작업 시작 → 중간 수정 요청 감소 → 일관된 결과물. 프리랜서는 소통비용 절감+포트폴리오 퀄리티↑. 클라이언트는 브랜드를 처음 언어화하는 경험. 양쪽 모두 재사용 동기 발생.

**팀 투표 메모**: 준형 5점, 은지 3점, 지현 5점, 윤팀장님 +1. "직접 겪는 문제라 페인포인트 실재성 확실. 다만 시장이 좁고 인터뷰 대상 섭외 어려움. 첫 스터디보단 이후 단계에 적합하다는 의견도."

## 2. 알라딘(서점) 앱 개선 (최종 투표 11점)

**도메인**: 독서·커머스 | **방향**: 리뉴얼

**고도화되어야 하는 이유 6가지**
1. **복잡한 비즈니스 구조 vs UX 미스매치** — 신간판매+중고매입/판매+굿즈+이벤트가 얽힌 복합 플랫폼인데 앱에서 정리가 안 됨
2. **신규 유입 약함** — 중고책 헤비유저는 두터우나 20대 초반 이하에게 "책 사는 앱" 이미지 자체가 없음
3. **개인화 엔진 사실상 없음** — 구매이력/찜/리뷰 데이터는 있는데 추천 알고리즘 미연결 (기회손실)
4. **중고거래 UX가 당근마켓 대비 열세** — 실시간검색·상태확인·거래흐름에서 밀림
5. **독서 라이프스타일 앱으로 확장 여지** — 독서기록·커뮤니티·작가팔로잉·읽기목표 추가 시 체류시간↑. 리디/밀리의서재가 "콘텐츠 소비"라면 알라딘은 "소유하는 독서 문화"로 포지셔닝 가능
6. **오프라인-온라인 단절** — 오프라인 매장 경험(책 고르는 재미, 중고 직매입)이 앱과 연결 안 됨. 재고 실시간 확인·매장픽업·매장전용 이벤트 같은 O2O 부재

**팀 투표 메모**: 준형 3점, 은지 4점, 지현 4점. "비주얼·인터랙션 관점에서 흥미로운 결과물 가능성. 페인포인트 분석은 수월하나 개선 방향이 모호하고 차별점 불분명."

## 3. 롯데시네마 앱 개선 (브레인스토밍 단계, 투표 진출 안 함)

**유형**: 개선 | 작성: 이준형

**문제점 5가지**
1. **비주얼/톤 일관성** — 브랜드 컬러 레드·핑크 혼용, 라인 아이콘 단일 사용으로 시각적 무게감·가독성 저하
2. **홈 화면 위계** — 캐러셀 3~4개 반복, 메인/서브 콘텐츠가 동일 크기·형태로 나열되어 시선 분산
3. **바텀 탭 구조** — 가장 중요한 예매 기능이 3번(중앙) 위치, 1번 자리를 부가기능(바로팝콘)이 차지, 마이페이지·메뉴가 별도 탭으로 분리되어 유사기능이 탭 2개 차지
4. **이벤트 진입장벽** — 이벤트 카드가 포스터/스틸컷 위주라 클릭 전까지 혜택 파악 불가
5. **개인화 부재** — 취향 기반 영화 추천 없음, 예매 단계 인디케이터 없어 흐름 파악 어려움

## 4. 자취러들을 위한 생활관리 플래너 (최종 투표 12점)

**도메인**: 자취·생활관리 | **방향**: 대체서비스

**개요**: 혼자 사는 사람이 놓치기 쉬운 식재료·생필품·청소 관리를 AI로 자동화하고, 자연스러운 구매 유도로 커머스 연계하는 생활 밀착형 앱.

**핵심 기능 3모듈**
- 🥬 **식재료 관리**: 구매일+유통기한 입력 → 소비기한 자동계산. 카테고리별 권장 소비기한 자동입력. 보유재료 기반 레시피 추천(일반/다이어트/탄단지). 유통기한 임박 알림. → 커머스: 식재료 구매
- 🧴 **생필품 관리**: 구매주기 2~3회 기록 → AI가 소비패턴 학습, 소진 예상일 며칠 전 알림 ("화장지 구매할 때 된 것 같아요"). → 커머스: 생필품 구독/구매
- 🧺 **청소 관리**: 세탁 가능 요일+환경(세탁기 유무 등) 설정 → 날씨·일정 고려해 최적 청소일 제안 ("이번 주말 날씨 맑아요, 이불빨래 어떠세요"). → 커머스: 청소용품 구매

**수익모델**: 커머스 연계 커미션 / 생필품·식재료 브랜드 광고 제휴 / AI레시피·상세분석 구독 프리미엄 / 기숙사·원룸단지 B2B 확장

**차별화 포인트**: 패턴 학습(몇 번 기록만으로 소비주기 자동분석) / 날씨·일정 연동 알림 / 광고가 아닌 "필요한 시점" 추천으로 전환율 극대화

**팀 투표 메모**: 준형 4점, 은지 3점, 지현 5점. "공감도는 높으나 기능 범위가 광범위해 스터디 기간 내 검증 어려울 수 있음. 지현은 VOC 접근성 면에서 높게 평가."
