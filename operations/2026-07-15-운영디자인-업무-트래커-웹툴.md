# 운영디자인 업무 트래커 (4단계 칸반 웹 도구)

## 핵심 요약
이벤트 페이지 운영디자인 업무 프로세스(기획안 수령·디자인 → 퍼블리싱 전달 → 심의필 대기 → 운영계 반영)를 한눈에 관리하기 위한 드래그앤드롭 칸반 보드 웹 도구를 제작. 로컬 브라우저에서 바로 사용 가능하며 데이터는 localStorage에 저장됨.

## 상세 내용

### 요청 배경 (운영디자이너 업무 프로세스)
1. 기획안 수령 및 디자인 (상단 이미지/하단 레이아웃)
2. 퍼블리싱 담당자에게 전달
3. 심의필 완료 대기
4. 이벤트 당일 운영계 반영 및 정상 작동 확인

### 구현된 기능
- **4단계 칸반 보드**: 각 단계별 카드 수 실시간 표시
- **드래그 앤 드롭**: 카드를 마우스로 끌어 단계 이동
- **심의필 확인 경고**: "운영계 반영" 단계로 이동 시 "심의필은 확인하셨나요?" 확인 모달 — 확인해야만 이동 완료
- **이벤트 추가/삭제**: 상단 입력창에 이름 입력 후 Enter 또는 버튼으로 카드 생성, 카드 우측 상단 ✕로 삭제
- **데이터 유지**: localStorage 저장으로 브라우저 재시작 후에도 유지
- UI는 화이트 톤 기반의 미니멀 디자인

### 활용
반복되는 이벤트 운영 프로세스 관리에 바로 재사용 가능한 내부 도구. 유사한 워크플로 관리가 필요할 때 템플릿으로 참고 가능.
