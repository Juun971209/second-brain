# KT — "KT 패밀리박스 5.0" 디자인시스템 프로젝트 (신규 클라이언트 발견) ⭐

> 출처: Google Drive 전수조사 — "Total/2025/_____KT 디자인 시스템/01 Data/KT 패밀리박스 5.0"
> 기존 메모리/대화기록에 전혀 없던 완전 신규 클라이언트 발견
> 정리일: 2026-07-03

## 확인된 사실

포트폴리오 루트 폴더 구조상 KANU/Nucare/CWNU와 동급인 **"04_KT Visual Communication"** 슬롯이 존재하고, 실제 작업 아카이브에서 **"KT 패밀리박스 5.0"**이라는 구체적 프로젝트가 확인됨. Junction의 표준 14단계 프로세스(01 Data~14 Application)를 동일하게 거친 정식 프로젝트로, KANU/DiGi LOCA와 같은 급의 완성도로 진행됨.

## 프로젝트 성격

**"KT 패밀리박스"** — KT의 서비스/앱으로 추정 (가족 결합 요금제 관련 서비스일 가능성). 이번 작업은 **컴포넌트 단위 디자인시스템 문서화** 프로젝트.

## 확인된 디자인시스템 문서 목록 (컴포넌트별)

- General_Color.pdf — 컬러 시스템
- General_Font.pdf — 폰트 시스템
- General_Character.pdf — 캐릭터/마스코트 가이드
- General_IMG.pdf — 이미지 가이드
- General_BTN.pdf / BTN_Other Case.pdf — 버튼 컴포넌트
- General_BTN_Floating BTN 상세가이드.pdf — 플로팅 버튼 상세
- General_Popup_기본형.pdf / 기본형+ICO BTN.pdf — 팝업 기본형
- General_Popup_Bottom sheet.pdf — 바텀시트
- General_Popup_Tooltip.pdf — 툴팁
- General_Popup_Toast Popup.pdf — 토스트 팝업

→ 실무 디자인시스템 문서로서 완성도가 매우 높음. 컴포넌트별로 세분화된 정식 스펙 문서 다수.

## 함께 발견된 것 — 확인 필요 (신한은행)

같은 "01 Data" 폴더 안에 아래 항목들이 섞여 있었음:
- 신한은행 철학 수립 장표
- 신한은행 pptx 참고
- _신한은핸 참고 자료

⚠️ **불명확**: 이게 (A) 신한은행도 별도 클라이언트였는지, 아니면 (B) KT 프로젝트 진행 중 벤치마킹/레퍼런스로 신한은행 자료를 참고한 것인지 확인 안 됨. 폴더명에 "참고"라는 단어가 붙은 것들이 많아 후자(레퍼런스)일 가능성이 높아 보이나, "철학 수립 장표"는 실제 작업물처럼 보이기도 함 — **준형 확인 필요**.
