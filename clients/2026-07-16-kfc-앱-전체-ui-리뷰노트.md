# KFC 앱 전체 UI 리뷰노트

## 핵심 요약
팀원(준형·은지·남지현)이 KFC 앱 전체 화면(로그인~마이~쿠폰~멤버십/카드~Foundation~Components)을 직접 사용하며 화면별로 남긴 코멘트를 정리한 Notion 문서. 전체 화면 캡처/원본은 피그마 UI 인벤토리("운영팀 스터디_vh")에서 확인 가능. 기존에 저장된 "인앱 메시지 용어" 메모(2026-07-15)보다 훨씬 포괄적인 전체 화면 리뷰 기록.

## 상세 내용

### 로그인
- 로그인 버튼 아래 여백 어색함 / 간편(SNS) 로그인 부재 / 회원가입 정보입력 텍스트 상하 간격 불일치

### 홈화면 (메인 · 장바구니)
- 상단 로고와 이벤트배너 로고 위치 중복 우려, 하단 플로팅 바코드 버튼 용도 불분명
- 장바구니 메뉴 바로삭제 기능 없음, 매장정보 없을 시 강제 이동

### 스마트오더/딜리버리
- 신규주소 등록: "배달받을 주소 입력"과 "새로운 주소"가 중복, 주소 저장 기능 부재(VOC와 동일 이슈)
- 기존주소: KFC 지정 매장에서만 배달 가능(VOC 동일), 팝업에 취소·X버튼 중복
- 메뉴선택: 탭별 아이콘 스타일 불일치, 버거 주문 뎁스가 길어짐(VOC 동일), **치킨나이트 프로모션 페이지(21~22시)에 이벤트 시간·주문방법 안내 부족**
- 결제: 네이버페이·카카오페이 등 주요 간편결제 수단 부재(KFC PAY·PAYCO·L.pay만 제공)
- 주문완료: 배달원 실시간 위치 파악 불가, 지도에 매장·배달주소 동시 표시 필요(쿠팡이츠 참고), 완료 애니메이션 완성도 아쉬움

### 스마트오더/징거벨오더
- 매장확인: 비활성 버튼 색상 불일치, 지도에 전체 매장 미표시, 마이>매장찾기 지도와 데이터 상이
- 메뉴선택: 상단 배너 카피("이제 줄 서지 마세요! 딩동~")가 전체 톤과 불일치, 메뉴 확인 진입 과정 과도하게 복잡(VOC 동일)
- 주문완료: 주문취소 버튼이 실제 취소 없이 안내만 표시 — 버튼 존재 이유 의문

### 쿠폰
- 매장방문: "모두 받기"와 개별 발급 차이 불분명, 할인가 강조 부재, 유효기간 표기 방식 개선 필요("D-일 남음"), 쿠폰에 "사용하기" 버튼 필요
- 징거벨오더: 할인율보다 실가격 강조 필요, 스마트오더용 쿠폰에 바코드 필요성 의문, 쿠폰 종류 구분이 탭으로만 되어 있어 혼동 우려

### 멤버십/카드
- 카드잔액 동전 회전 애니메이션 어색, 카드 스캔 버튼 가시성 부족, 이용내역 기간 탭 크기 어색, 리스트 정렬 기준 불일치

### 마이
- 상단 아이콘 식별성 낮음, 등급 캐릭터 일러스트(달걀·병아리) 목적 불분명, 아이콘 스타일 전반 불일치(브랜드 정체성 약화)
- 설정 페이지 별도 존재 필요성 의문
- 등급 배지 컬러가 탭마다 다름, Welcome 쿠폰 아이콘 어색
- 이벤트 상세 가독성 낮음(약관처럼 읽힘)
- 매장찾기: 검색결과 레이아웃 오류, 검색 UI 불일치, 팝업 매장 별도 카테고리 필요, SNS 공유 링크 업데이트 안 됨
- 고객센터: 상단 일러스트 문맥 불일치, 문구 줄바꿈 어색, 카테고리 그리드 여백/정렬 문제, FAQ 버튼 라운드값 불일치

### 마케팅수신동의 · 메인배너 · 주문현황 · 광고
- 위치정보 동의 팝업 문구 모호, 조리중 완성 예상시간 미제공(타 앱 대비), FAQ 화면 광고 노출 필요성 의문

### Foundation (스포이드 채취, 참고용)
- 컬러: 그레이스케일 11단계로 과다, 레드 계열 4종 혼용
- 타이포: 여기어때 잔난체2(강조) / 산돌 Gothic Neo(본문)
- 아이콘: 레드박스·이모지풍·라인드로잉 혼재로 중구난방
- Radius: 각진 스타일 기본이나 부분 혼용, 기준 모호
- Space: 마이 메인 좌우마진 15px(애매), 팝업마다 마진값 제각각

### Components
버튼(Solid/Outline/Disabled), 바텀 네비게이션, 체크박스, 디바이더, 리스트, 인디케이터, 드랍다운, 로딩, 팝업, 라디오버튼, 스위치, 탭, 입력창, 태그, 검색창, 탑 네비게이션 — 전체 화면에서 쓰이는 컴포넌트 유형 인벤토리 정리됨.
