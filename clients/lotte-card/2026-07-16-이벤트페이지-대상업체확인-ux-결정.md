# 롯데카드 이벤트 페이지 — '대상업체 확인' 노출방식 UX 결정

## 핵심 요약
갱신유도/갱신완료 이벤트 페이지에 별도로 노출되는 "대상업체 확인" 영역의 표시 방식을 두고 팝업(레이어)안과 아코디언(인라인 펼침)안 2종을 검토. 롯데카드 자사 이벤트 페이지의 "국가별 대상 호텔 확인" 버튼(클릭 시 페이지 내 인라인으로 표가 펼쳐지는 방식)을 실제 벤치마크로 확인하고 아코디언안을 2안으로 제안. 클라이언트(인경혜책임)에게 전달할 확인 요청 메시지를 간결하게 다듬는 과정도 함께 진행. 별도로, 외부에서 공유받아 원본 파일이 없는 배너에 대한 수정 불가 안내 문구도 정리.

## 상세 내용

### UX 검토
- 롯데카드 실제 이벤트 페이지의 "국가별 대상 호텔 확인" 버튼 확인 결과: 별도 팝업 레이어가 아니라 버튼 아래 페이지 스크롤 흐름 안에서 표가 인라인으로 펼쳐지는 "아코디언(펼침형 인라인 리스트)" 방식임을 확인.
- 이 사례를 2안(아코디언 노출)의 실제 벤치마크 레퍼런스로 채택.

### 클라이언트 전달 메시지 (최종본)
> 인경혜책임님
> 안녕하세요.
> 이벤트 페이지 디자인 확인 부탁드립니다.
>
> 이벤트 페이지는 갱신유도, 갱신완료 화면과 대상업체 확인 영역으로 구성되어 있습니다.
> 대상업체 확인 노출 방식은 2안으로 제안드립니다.
>
> 1안: 팝업 노출 (기획안 반영)
> 2안: 아코디언 노출 (버튼 클릭 시 인라인 펼침, 롯데카드 벤치마크 참고)
>
> 확인 부탁드립니다.
> 감사합니다.

### 배너 수정 관련 안내 (참고)
외부에서 공유받은 배너 이미지에 대해 다음과 같이 정리: "해당 배너는 저희가 제작한 것이 아니라 외부에서 공유받은 이미지여서, 수정이 불가능할 것 같습니다." (원본 작업 파일/에디터블 소스 미보유 사유 포함)
