## 요약

팀원(준형·은지·지현)이 KFC 앱 화면을 직접 살펴보며 화면별로 남긴 코멘트 기록. 전체 화면 캡처는 Figma UI 인벤토리에서 확인 가능.

## 로그인 화면

**화면 흐름**
마이(MY KFC) 진입 → 로그인 → 회원가입(약관 동의) → 회원가입(정보 입력)

**발견된 이슈**

| 화면 | 이슈 | 발견자 |
| --- | --- | --- |
| 로그인 | 로그인 버튼 아래 여백이 어색함 | 은지 |
| 로그인 | 간편 로그인 기능 부재 (SNS 로그인 등 없음) | 준형 |
| 회원가입 · 정보입력 | 텍스트 상하 간격이 항목마다 안 맞음 | 은지 |
